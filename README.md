# AI-Powered Predictive Maintenance (Virtual Simulation)

## Overview

This project simulates an AI-based predictive maintenance system using IoT sensor data.

## Features

* Simulated sensor data (temperature, vibration, current)
* Machine Learning model (Random Forest)
* Flask API for real-time predictions
* Failure detection system

## How to Run

1. Train Model
   python src/train_model.py

2. Run API
   python src/api.py

3. Run Simulation
   python main.py

## Output

* Real-time sensor data simulation
* AI-based prediction (Failure / Normal)

## Tech Stack

Python, Pandas, Scikit-learn, Flask
